# Gestion-des-tudiants---Backend-Spring-Boot
Gestion des Étudiants - Backend Spring Boot Description  Ce projet est une API REST développée avec Spring Boot permettant de gérer les étudiants à travers des opérations CRUD (Create, Read, Update, Delete).  L'application expose des endpoints REST qui peuvent être consommés par une application frontend  afin de gérer une liste d'étudiants.
